AnimeGif API Console

Integrate high-quality anime visuals into your apps using our simple REST API.

Base URL: https://animegif-ten.vercel.app/api/get-waifu

1. Requesting Data

You can request images in two ways: Single (one URL) or Collection (30 URLs at once).

Method Path Result
GET ?type=sfw&cat=happy 1 Random Image URL
POST ?type=sfw&cat=dance Array of 30 Image URLs

2. Understanding the Output

The API always returns a JSON object. Here is what you will see:

// If you used GET (Single) { "url": "https://api.waifu.pics/sfw/happy/392.gif" } // If you used POST (Collection) { "files": [ "https://api.waifu.pics/sfw/dance/1.gif", "https://api.waifu.pics/sfw/dance/2.gif", ... // 28 more links ] }

3. Implementation Examples

JavaScript (Web Apps)

async function getAnime() { const response = await fetch('https://animegif-ten.vercel.app/api/get-waifu?type=sfw&cat=smile'); const data = await response.json(); console.log(data.url); // Use this link in your img tag }

Python (Bots & Scripts)

import requests url = "https://animegif-ten.vercel.app/api/get-waifu" params = {"type": "sfw", "cat": "waifu"} # For a single image response = requests.get(url, params=params) print(response.json()['url']) # For a collection response = requests.post(url, params=params) print(response.json()['files'])
