AnimeGif API Docs

The AnimeGif API is designed for high-speed delivery of curated anime visuals. Integrate our endpoints directly into your application using the documentation below.

For technical support or bug reports, please visit our repository on GitHub.
Always use the /api/get-waifu route to ensure your requests are proxied through the AnimeGif secure layer.

Get Single Image

Receive one direct image stream from your chosen category.

/api/get-waifu?type=type&cat=category GET

Valid types: sfw, nsfw.

Get Collection (Many)

Receive an array of unique image URLs.

/api/get-waifu?type=type&cat=category POST
